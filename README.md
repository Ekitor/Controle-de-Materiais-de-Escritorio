# Controle-de-Materiais-de-Escritorio
Sistema web desenvolvido para otimizar o controle de saída e distribuição de materiais de escritório da empresa OpusHub, reduzindo a burocracia e falhas sistêmicas.

# Sistema de Controle de Materiais de Escritório - OpusHub

# Sobre o Projeto
Este projeto consiste no desenvolvimento de um sistema de controle de materiais de escritório para a empresa OpusHub, realizado no âmbito acadêmico do Instituto Federal de Pernambuco (IFPE) - Campus Jaboatão dos Guararapes. 

O sistema tem como objetivo principal solucionar problemas recorrentes de quedas de sistema e excesso de burocracia. A plataforma visa facilitar a coordenação de patrimônio, dinamizar o controle de saída e distribuição de materiais, e oferecer uma interface intuitiva para o treinamento de novos funcionários da administração. O desenvolvimento da interface e da lógica das operações de CRUD foi construído utilizando HTML, CSS, JavaScript, integrado a um banco de dados estruturado em SQL.
